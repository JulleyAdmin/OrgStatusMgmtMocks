Welcome to AI Org Platform

This is a reusable navigation shell component. The sidebar can be collapsed on desktop and transforms into a drawer on mobile devices.

Features

  • Collapsible sidebar with smooth animations
  • 13 navigation menu items with icons
  • Active state styling with gradient background
  • Responsive mobile drawer with backdrop
  • Top navigation with search, notifications, and profile menu
  • Tailwind CSS utility classes for easy customization

Usage Instructions

To use this navigation shell in your screens:

  1. Include this HTML file or copy the navigation structure
  2. Update the active state for the current page
  3. Replace the main content area with your screen content
  4. Ensure Tailwind CSS CDN is included or use your build process
  5. Test responsive behavior on different screen sizes
